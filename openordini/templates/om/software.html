Questo è lo stack software utilizzato nel backend:

  • apache - web server, con mod_wsgi, per gestire processi Python
  • django - web framework, con numerosi moduli e componenti sviluppati da terze parti
  • postgres - database management system
  • solr - indicizzazione testi e servizi di ricerca
  • tomcat - servlet container per solr
La distribuzione GNU-Linux è la Debian Squeeze.

Per lo sviluppo lato client, questi i software usati:

  • jQuery
  • jQueryUI
  • bootstrap
  • fontawesome
